# CodeAlpha Python Internship Projects

Hello!
This repository highlights the projects I completed during my Python Programming Internship at CodeAlpha. During this period, I focused on strengthening my core Python skills, enhancing problem-solving abilities, and understanding how simple programs can be applied to real-world scenarios.


## Projects Completed

### Hangman Game

This is a fun, text-based game in which the player tries to guess a hidden word one letter at a time, with a limited number of attempts available.

**What I learned:**
I practiced working with strings, lists, loops, and the `random` module while building this mini game.


### Stock Portfolio Tracker

This project calculates the total value of stocks using user input along with predefined stock prices.

**What I learned:**
I gained experience using dictionaries, handling user input, and performing basic calculations. It also gave me insight into how simple programs can be useful in finance-related tasks.


### Chatbot

This project is a simple rule-based chatbot that interacts with users through predefined responses. It can handle basic inputs such as greetings and common questions.

**What I learned:**
I learned to use conditional statements (`if-elif`), loops, and functions to build a basic conversational application.
