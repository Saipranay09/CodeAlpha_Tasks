## Chatbot

This project is a simple rule-based chatbot built with Python. It communicates with users by responding to predefined inputs, making it an excellent introduction to basic conversational logic.

### Features

* Responds to greetings such as "hello" and "hi"
* Handles basic user queries
* Provides preset responses
* Supports interactive command-line conversations

### Concepts Used

* `if-elif` conditional statements
* `while` loops for continuous interaction
* Functions for better code organization
* Basic string manipulation

### How to Run

* Ensure Python is installed on your system
* Execute the following command in the terminal:
  `python chatbot.py`

### Example Interaction

You: hello
Bot: Hi there! How can I help you?

You: how are you
Bot: I'm just a bot, but I'm doing great!
