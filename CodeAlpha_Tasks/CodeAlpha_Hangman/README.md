## Hangman Game

This is a simple terminal-based Hangman game where users attempt to uncover a hidden word by guessing one letter at a time.

## Features

* Chooses a random word for each round
* Limits the number of guesses a player can make
* Displays the current state of the word after each attempt

## Concepts Used

* `random` module
* Loop structures
* Working with strings and lists

## How to Run

python hangman.py
