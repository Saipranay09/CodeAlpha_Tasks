## Stock Portfolio Tracker

This project is built to calculate the total investment value based on user input and predefined stock prices.

## Features

* Enables users to input stock names along with their quantities
* Computes the total value of the investment
* Easy to understand and user-friendly

## Concepts Used

* Dictionaries
* Input and output handling
* Basic mathematical operations

## How to Run

python stock_tracker.py
